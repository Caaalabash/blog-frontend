# vue-blog

## Start

````
git clone https://github.com/Caaalabash/vue-blog.git

// 前端服务
cd frontend
npm install --registry https://registry.npm.taobao.org
npm run local

// 后端服务, 需保证mongodb以及redis的连接
cd backend
npm install --registry https://registry.npm.taobao.org
npm run local
````

## Include

+ FrontEnd

  + Vue-cli3

  + Vue + Vue-router + Vuex + axios

  + ElementUI

  + Less

  + PWA

+ BackEnd

  + express

  + mongoose

  + redis

  + socket

+ Build & Deploy

  + webpack

  + docker

  + travis
