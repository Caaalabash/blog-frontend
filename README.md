## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

## 起步
+ 进入项目目录后安装依赖
````
 npm install
````
+ 进入server目录安装依赖
````
 npm install
````
+ 在有mongodb的条件下启动服务器
````
  node bin/www
````
+ 回到项目目录启动前端
````
  npm run dev
````
