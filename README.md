# vue-blog [![Build Status](https://travis-ci.org/Caaalabash/vue-blog.svg?branch=publish)](https://travis-ci.org/Caaalabash/vue-blog)

## Start

````
git clone https://github.com/Caaalabash/vue-blog.git

npm install

npm run local
````

## Include

+ FrontEnd

  + Vue-cli3

  + Vue + Vue-router + Vuex + axios

  + ElementUI

  + Less

  + PWA

+ BackEnd

  + express

  + mongoose

  + redis

  + socket

+ Build & Deploy

  + webpack

  + docker

  + travis
